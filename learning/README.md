# Learning in Gigastep

## MARLlib

### Installation
```
$ cd MARLlib
$ pip install -e .
$ pip install ma_gym
```

### Run
```
$ python examples/train_gigastep.py
```

## evosax

### Installation
```
$ pip install evosax
$ pip install tqdm
```

### Run
```
$ python3 learning/evosax/scripts/run_open_es.py
```

### Notes
Evosax is under development and some features may move / not be synced across functions.
